# 4 repositorios de GitHub que todo JavaScript Developer debe conocer

Aprender programación puede ser una de las áreas mas increíbles para aprender ya que desde el día 0 puedes ponerte manos a la obra y poner en práctica todo el conocimiento que vas adquiriendo. Muchas veces lo que hace falta es conocer un contenido donde podamos tener la información que necesitamos y que nos permita prácticar. Por eso hoy te traigo una recopilación de repositorios donde podras aprender sobre JavaScript. 

## 33 Concepts Every JavaScript Developer Should Know  

En este repositorio podrás encontrar una lista de 33 conceptos que los desarrolladores JavaScript debe conocer. Este no solo incluye la lista de conceptos sino que también por cada concepto puedes tener una lista de articulos recomendados donde podrás obtener mas información de cada uno de conceptos. Esta es mucha información para digerir en un día, te recomiendo que leas rápidamente cada uno de los artículos listados y escoge 2 artículos que sientas que comprendas mas facilmente. 

https://github.com/leonardomso/33-js-concepts

## Algoritmos y Estructuras de Datos en JavaScript

Este es un repositorio donde podrás poner en práctica todos tus conocimientos. Si quieres empezar a entender los fundamentos de la ciencia de la computación este es un gran repositorio para emepzar ya que este te brinda las pruebas para verificar si tu implementación es correcta. Tambien incluye una clasificación si el reto es recomendado para principiantes o avanzados  y un tutorial de como correr las pruebas. 

https://github.com/trekhleb/javascript-algorithms/blob/master/README.es-ES.md

## Web Development for Beginners - A Curriculum

Este es un recurso creado por Microsoft donde podrás encontrar lecturas y tutoriales para crear distintas aplicaciones usando solamente JavaScript. Puedes usarlo localmente y puedes ver el contenido en diferentes formatos. 

https://github.com/microsoft/Web-Dev-For-Beginners
